Naturebook

Record wildlife sightings!

App designed to encourage exploration of nature by younger kids.

Tech used: React frontend, Ruby on Rails backend, Material UI and custom CSS for styling, Google Maps API

Functionality:

  • Create a profile page and record wildlife sightings.
  • See a list of wildlife sightings from all other users and consult a map with pins dropped at the latitude and longitude of recorded sightings.
  • Contribute to an in-app field guide, by adding organisms as they find them.
  • Collect "badges" based on how many organisms of a particular group they record.
